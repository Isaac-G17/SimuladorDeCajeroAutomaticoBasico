# SimuladorDeCajeroAutomáticoBásico
Proyecto de consola que simula cajero automático con funcionalidades esenciales.

# Autor

Este proyecto fue desarrallado por **Isaac Guzman**

## Funcionalidades

El sistema permite realizar las siguientes operacion:
 - Consultar saldo 
 - Retirar dinero
 - Depositar dinero
 - Validación de montos invalidos
 - Mensaje cuando la opción ingresada es incorrecta

## Conocimientos usados

 - Fundamentos de Python
